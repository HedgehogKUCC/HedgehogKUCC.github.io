Step 2 - 安裝 hexo-CLI

可以讓 hexo 使用指令的工具

Mac 前面會需要加上 sudo

  • sudo : 代表給予最高權限來執行

Step 3 - 快速建立雛型

在桌面建立 資料夾


Step 4 - 啟動本地端預覽

s = server , localhost:4000

如果 localhost:4000 不能使用

請下指令 hexo server -p 5000

將 port 改為 5000 即可


Step 5 - 建立 (編譯) 靜態網站


Step 6 - 開始版本控制


在 Github 新增 repository

將 repository 名稱設置為 {Github用戶名稱}.github.io

  • 每個帳戶只有一個 , 能放在最根目錄的專案 .
  • repository 名稱請都設為 小寫英文

先將 本地端資料夾 連上遠端 Github


Step 7 - 修改 _config.yml

使用 SSH 要記得設定金鑰 !!!

  • 設定 Github SSH 金鑰

branch 要設定的是 master

部署在 Github Page 要特別注意在這個網域之下 , master 是預設為發佈的 branch .


Step 8 - 開始部署

install hexo 自動部署工具

可以至 Github —> Settings

往下拉至 GitHub Pages 就可以看到發佈的網址 ~


Step 9 - 開啟分支 (branch)

推上 Github
